Who is this for? 

Computer Science or related field students which takes same/similar subjects. 

 

What painful problem does it solve? 

Finding answers and solutions to their subject related question and problems easier. Creating an archive of materials online which is essential for students. Being in direct connection with other students, graduates and professors. 

 

Why is your solution better/simpler? 

It gives all the opportunity to any student asking their question to a whole community of people who have been through similar experience. Faster problem solving community. 

 

MVP: 

A working university Q&A forum where students can ask questions, get answers, vote on quality content, and search for information by subject. (Milestone 1-7) 

 

Milestone List: 

 
Milestone 1: Foundation 

Goal: Project runs, database connected, basic setup done 

Deliverables: 

    Spring Boot project created 

    PostgreSQL database setup 

    Basic project structure (controllers, services, repositories) 

    Database schema designed 

    React project created 

    Projects talk to each other (test endpoint works)
